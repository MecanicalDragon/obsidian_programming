Виртуальные потоки — это **лёгкие потоки**, которые выполняются поверх ограниченного пула настоящих (platform) потоков, называемых в данном контексте **carrier threads**. Когда виртуальный поток блокируется на I/O, он **паркуется** (снимается с carrier) и не тратит системный поток, освобождая его для других задач.
